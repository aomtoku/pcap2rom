pcap2rom for AXI stream bus
=================================================

Usage : 
  $ ./pcap2rom.pl -r <Pcap File> -w <Output ROM file> \
       -p <Inter Packet GAP (# of clock cycles)>      \
       -f <Inter Flit GAP (# of clock cycles)>
